# Guide DIY - Kiosque Compact Station Blanche

## 🎯 Objectif

Fabriquer un kiosque compact et économique pour la Station Blanche avec un écran plus petit (10-13 pouces) au lieu des 21 pouces standards.

**Avantages format compact :**
- ✅ Moins encombrant (bureau, comptoir)
- ✅ Coût réduit (écran + matériaux)
- ✅ Plus mobile/déplaçable
- ✅ Moins intimidant pour l'utilisateur
- ✅ Fabrication DIY simple

## 🛠️ Plans de Fabrication Détaillés

### Modèle 1 : Totem Bureau Compact (13")

#### Étape 1 : Découpe des pièces

**Liste de découpe (MDF/Contreplaqué) :**
```
- Base : 25x20x2cm (MDF 18mm)
- Panneau arrière : 40x20x1cm (Contreplaqué 12mm)
- Support PC : 20x15x1cm (Contreplaqué 12mm) x2 pièces
- Capot PC : 20x15x1cm (Contreplaqué 12mm)
```

#### Étape 2 : Assemblage structure

1. **Fixer panneau arrière sur base**
   - Pré-percer 4 trous Ø4mm
   - Visser avec vis bois 4x40mm
   - Renforcer avec équerre métal (optionnel)

2. **Créer compartiment mini PC**
   - Fixer les 2 supports latéraux à 12cm de la base
   - Espacement : largeur du mini PC + 2cm
   - Laisser ouvertures pour ventilation

3. **Installer fixation VESA écran**
   - Platine VESA 75x75mm fixée à 25cm du haut
   - Utiliser vis M4 (écran) et vis bois 4x20mm (structure)
   - Vérifier niveau avant fixation

#### Étape 3 : Passages câbles

```
Perçages nécessaires :
- Ø15mm dans base (câble alimentation)
- Ø10mm dans support PC (HDMI + USB)
- Goulotte verticale arrière (passe-fil)
```

#### Étape 4 : Finition

1. **Ponçage**
   - Grain 120 puis 180
   - Arrondir les angles

2. **Peinture**
   - Sous-couche bois
   - 2 couches peinture acrylique
   - Couleurs Owlcub : Vert (#10b981) + Blanc

3. **Montage final**
   - Fixer mini PC dans compartiment (velcro ou vis)
   - Connecter écran (HDMI + USB touch)
   - Passer câbles dans goulottes
   - Fixer cache arrière (optionnel)

#### Schéma de montage détaillé

```
Vue éclatée :

     [Écran 13"]
         ↓
    Fixation VESA
         ↓
  ┌─────────────┐
  │ Panneau     │
  │ arrière     │← Passage câbles HDMI/USB
  │             │
  │  ┌────┐    │
  │  │Mini│    │← Compartiment PC (ventilé)
  │  │ PC │    │
  └──┴────┴────┘
  ┌─────────────┐
  │    Base     │← Passage câble alim
  └─────────────┘
```

## 📝 Checklist Fabrication

### Avant de commencer

- [ ] Mesurer emplacement final (hauteur, largeur, profondeur)
- [ ] Vérifier compatibilité VESA écran choisi (75x75 ou 100x100)
- [ ] Acheter tous les matériaux
- [ ] Préparer espace de travail (établi, protection)

### Fabrication

- [ ] Découper toutes les pièces aux dimensions
- [ ] Poncer toutes les surfaces
- [ ] Pré-percer tous les trous de vissage
- [ ] Assembler structure à blanc (sans colle)
- [ ] Vérifier stabilité et alignement
- [ ] Démonter et appliquer finition (peinture)
- [ ] Assemblage final une fois sec
- [ ] Installer fixation VESA et mini PC
- [ ] Tester stabilité avec poids

### Installation & Tests

- [ ] Positionner à l'emplacement final
- [ ] Fixer écran tactile (VESA)
- [ ] Installer mini PC dans compartiment
- [ ] Connecter tous les câbles (HDMI, USB, alim)
- [ ] Démarrer et tester tactile
- [ ] Installer logiciel Station Blanche
- [ ] Test scan USB complet
- [ ] Fixer cache arrière (finition)

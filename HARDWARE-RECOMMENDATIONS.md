## 📐 Plans de Fabrication Custom (DIY)

### Totem Sur Pied - Spécifications
```
Dimensions :
- Hauteur totale : 140 cm (écran à hauteur yeux)
- Base : 40x40 cm (stable)
- Profondeur : 15-20 cm
- Poids : 15-20 kg

Matériaux :
- Structure : Tube acier 40x40mm
- Habillage : Tôle aluminium 2mm ou bois contreplaqué 12mm
- Finition : Peinture époxy ou vernis

Composants VESA :
- Platine VESA 100x100mm fixée à 130cm du sol
- Compartiment mini PC : 25x25x12cm (derrière écran)
- Passages câbles : Goulotte interne verticale

Fixation sol :
- 4 trous Ø10mm dans base (boulonnage optionnel)
- Ou lestage interne 10kg (sable, métal)
```

**Coût matériaux** : 150-300€
**Temps fabrication** : 1-2 jours (menuisier/métallier)
